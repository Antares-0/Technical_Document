# 项目说明

## 一、项目文件说明
1. `sft.py`：微调脚本，使用gpt2作为预训练模型，使用sst电影评论数据集进行微调（不使用标签，仅仅使用电影评论的句子进行微调）
2. `sft_test.py`：微调测试脚本，测试微调的结果，可以发现给定无偏好的提示词，返回会更偏向于返回电影相关的续写
3. `rm.py`：奖励模型定义脚本，定义了关于奖励模型的结构，主要是gpt2用于提取信息，在gpt2上增加了一个线性层，来实现奖励的映射
4. `rm_train.py`：使用电影评论数据集作为奖励模型的训练集，其中积极电影评论的奖励默认设置为1，用于训练奖励模型；同时在句子末尾添加了特殊字符，将特殊字符的隐藏层输出经过线性层处理后用于loss计算
5. `actor_critic_model.py`：演员评论家模型，gpt2-sft作为actor，critic是对应的线性层

## 二、项目说明
1. 项目整体作为PPO + RLHF的练习项目，融合了微调、奖励模型的训练、ppo强化学习对微调模型的人类偏好调整等技术
2. 使用gpt2作为基座模型进行微调，使用ppo算法用于强化学习微调
3. 注意奖励和价值是两个不同的概念，奖励是环境给予的；但是价值是评论家算法提供的，并不代表真实的奖励，而是一种奖励的单步模拟
