# Agent智能体

## 一、Agent基本概念
1. Agent：智能体，能够感知观景并且采取相应行为；Agent的核心逻辑是LLM根据环境动态变化，选择具体执行的行动，反过来影响环境，通过多伦迭代重复执行上述步骤，直到完成目标，总结就是PPA
   - P：感知Perception，从环境中收集信息
   - P：规划Planning，为了某一目标做决策的过程
   - A：行动Action，基于感知和规划作出的具体动作
