## 三、Go语言开发工具包
1. Go mod是什么？
   1. go mod是go官方推出的包管理工具，用于管理项目依赖，相当于java的maven和gradle
   2. go mod生成了两个文件
      1. go.mod 用于记录依赖
      2. go.sum 用于校验文件
   3. go mod对应的老版本是GOPATH，GOPATH
   4. Go mod常用命令：
      ```bash
      
      # 自动下载依赖
      go mod tidy
      ```
